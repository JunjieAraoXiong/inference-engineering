# Personal Notes

This folder is for your own durable understanding. Chapter notes follow the book order; concept notes cut across chapters.

Use this rule for every note:

- Explain the concept in your own words.
- Name the metric or signal that proves it matters.
- Include one failure mode.
- Include one production implication.

## Study Loop

1. Read one book section.
2. Update the chapter note.
3. Update or create one concept note.
4. Add one question or experiment.
5. Mark progress only when you can explain the tradeoff without looking.
